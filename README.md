### Installation

[![Installation](https://github.com/Vaporjawn/universities/actions/workflows/install.js.yml/badge.svg)](https://github.com/Vaporjawn/universities/actions/workflows/install.js.yml)

```bash
npm install
```

```bash

```

```bash
npm run dev
```
